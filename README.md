# AJ Distributors

Corporate website for AJ Distributors — a grocery distribution company serving supermarkets.

## Structure

```
AJ/
├── frontend/    # Next.js static site (current)
└── backend/     # API server (future)
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The site runs at [http://localhost:3000](http://localhost:3000).

### Build for Production

```bash
cd frontend
npm run build
```

Static HTML output is generated in `frontend/out/` and can be deployed to any static host.
